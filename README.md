## About

Peer-to-peer video calling app built with WebRTC. I built this during my internship prep to understand how real-time video communication works under the hood.

## Tech Stack

Node.js, WebRTC, Socket.io, Express

## Getting Started

```bash
git clone https://github.com/shashhii/Video-Conferencing-App.git
cd Video-Conferencing-App
```
